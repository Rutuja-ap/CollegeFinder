# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-07-10

### Added

* User authentication using NextAuth
* College discovery and search
* College comparison feature
* College predictor based on entrance exam rank
* Personalized saved colleges
* College-specific Q&A section
* Responsive user interface using Tailwind CSS
* PostgreSQL integration with Prisma ORM

### Improved

* Enhanced project documentation
* Added application screenshots to README
* Improved authentication security
* Updated environment variable configuration

### Fixed

* Minor UI improvements
* General bug fixes and performance optimizations
