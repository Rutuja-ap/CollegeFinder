# Contributing

Thank you for your interest in contributing to **CollegeFinder**.

We welcome contributions that improve the project. Before contributing, please follow the guidelines below.

## Getting Started

1. Fork the repository.
2. Clone your fork locally.
3. Install the dependencies.

```bash
npm install
```

4. Start the development server.

```bash
npm run dev
```

## Contribution Guidelines

* Create a separate branch for your changes.
* Write clean, readable, and well-documented code.
* Follow the existing project structure and coding style.
* Test your changes before submitting.
* Write meaningful commit messages.

## Pull Requests

* Clearly describe the purpose of your changes.
* Reference any related issues, if applicable.
* Ensure the project builds successfully before creating a Pull Request.

Thank you for helping make **CollegeFinder** better!
